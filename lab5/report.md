# Отчет по лабораторной работе №5

## Выполненные задания

В ходе лабораторной работы были реализованы следующие задачи:
1. **Разработка интерфейса управления финансами**: Создана веб-страница с формой для ввода транзакций (сумма, категория, описание) и таблицей для их отображения.
2. **Динамическое манипулирование DOM**: Реализовано добавление новых строк в таблицу без перезагрузки страницы, а также функционал удаления записей с автоматическим пересчетом общего баланса.
3. **Эффективная обработка событий**: Применено делегирование событий на уровне таблицы для управления кликами по строкам и кнопкам удаления, что оптимизирует использование памяти и упрощает работу с динамическим контентом.

## Контрольные вопросы

### 1. Каким образом можно получить доступ к элементу на веб-странице с помощью JavaScript?
Для этого используются методы объекта `document`:
- `getElementById('id')` — поиск по уникальному идентификатору.
- `querySelector('selector')` — поиск первого элемента, соответствующего CSS-селектору.
- `querySelectorAll('selector')` — получение всех элементов по селектору в виде статического списка NodeList.

### 2. Что такое делегирование событий и как оно используется?
Делегирование — это установка одного обработчика на родительский элемент вместо множества обработчиков на дочерние. Благодаря всплытию событий, родитель может поймать событие от любого потомка через `event.target`. Это экономит ресурсы и автоматически подхватывает новые (динамически добавленные) элементы.

### 3. Как можно изменить содержимое элемента DOM после его выборки?
- `textContent` — изменяет только текстовое содержимое (безопасно).
- `innerHTML` — позволяет вставлять HTML-разметку (требует осторожности из-за XSS).
- `value` — используется для изменения значения в полях ввода (input, textarea).

### 4. Как можно добавить новый элемент в DOM дерево?
1. Создать элемент: `const el = document.createElement('tag')`.
2. Настроить его (текст, классы, атрибуты).
3. Добавить в дерево: `parent.appendChild(el)`, `parent.prepend(el)` или использовать `insertAdjacentHTML()` для вставки готовой строки разметки.
