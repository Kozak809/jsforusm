# Контрольные вопросы

## 1. Какие методы массивов можно использовать для обработки объектов в JavaScript?

Для обработки массивов объектов часто используются следующие методы:

*   **forEach()**: Перебор всех элементов массива.
*   **map()**: Создание нового массива путем трансформации каждого элемента (например, извлечение конкретного свойства из объекта).
*   **filter()**: Создание нового массива, содержащего только элементы, удовлетворяющие условию.
*   **reduce()**: Агрегация данных массива в одно значение (например, суммирование полей объектов).
*   **find()**: Поиск первого элемента, удовлетворяющего условию.
*   **some() / every()**: Проверка, удовлетворяет ли хотя бы один или все элементы условию.
*   **sort()**: Сортировка массива (например, по значению свойства объекта).

## 2. Как сравнивать даты в строковом формате в JavaScript?

Даты в строковом формате можно сравнивать лексикографически (как строки), если они записаны в формате ISO 8601 (`YYYY-MM-DD` или `YYYY-MM-DDTHH:mm:ss.sssZ`). В этом формате компоненты даты идут от старшего к младшему, что обеспечивает корректное сравнение строк.

Пример:
```javascript
"2023-01-01" < "2023-01-02" // true
"2023-02-01" > "2023-01-01" // true
```

Если формат отличается (например, `DD.MM.YYYY`), строки необходимо преобразовать в объекты `Date` перед сравнением:
```javascript
new Date("01.01.2023").getTime() < new Date("02.01.2023").getTime()
```

## 3. В чем разница между map(), filter() и reduce() при работе с массивами объектов?

*   **map()**: Используется для **трансформации**. Возвращает новый массив **той же длины**, что и исходный, где каждый элемент является результатом применения функции к исходному элементу. (Пример: получить массив только имен из массива пользователей).
*   **filter()**: Используется для **выборки**. Возвращает новый массив, который может быть **меньше** (или равен) исходному по длине, содержащий только те элементы, для которых функция вернула `true`. Структура элементов не меняется. (Пример: получить список совершеннолетних пользователей).
*   **reduce()**: Используется для **агрегации**. Возвращает **одно значение** (которое может быть числом, строкой, объектом или массивом), полученное путем свертки всего массива. (Пример: посчитать общую сумму покупок).
